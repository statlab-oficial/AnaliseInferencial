2  Distribuições Amostrais

2.1 Introdução

O que é a estatística? Consiste em três grandes áreas:

  • Coleta de Dados: planos amostrais e desenhos experimentais.

  • Estatística Descritiva: resumos numéricos e gráficos dos dados coletados a partir de uma amostra.

  • Estatística Inferencial: estimação, intervalos de confiança e testes de hipóteses sobre parâmetros de interesse.

Os procedimentos estatísticos fazem parte (passos 2 a 5 abaixo) do Método Científico, inicialmente defendido por Sir Francis Bacon (1561-1626), que escreveu “aprender os segredos da natureza envolve coletar dados e realizar experimentos”.

A metodologia moderna:

  1. Observar algum fenômeno.
  2. Formular uma hipótese que explique o fenômeno.
  3. Coletar dados.
  4. Testar: Os dados suportam a hipótese?
  5. Conclusão. Se o teste falhar, volte ao passo 2.

“As estatísticas não mentem, mas os mentirosos usam estatísticas.” — Mark Twain.

2.1.1 O que é a Estatística Matemática?

É o estudo dos fundamentos teóricos da estatística.

O que é uma estatística?

Seja \(X_{1},X_{2},...,X_{n}\) um conjunto de variáveis aleatórias observáveis (como, por exemplo, uma amostra aleatória de \(n\) indivíduos de uma população de interesse). Uma estatística \(T\) é uma função aplicada a essa amostra: \[T=\mathcal{T}(X_{1},X_{2},...,X_{n})\]

População vs. Amostra

  • População: O grupo inteiro de indivíduos (sujeitos ou unidades), que pode ser existente ou conceitual, sobre o qual pretendemos obter informação.

  • Amostra: Uma parte da população a partir da qual os dados são recolhidos.

Parâmetro vs. Estatística

Parâmetro: Um valor numérico calculado a partir de todos os indivíduos na população. (Discreta / Contínua)

  • Média populacional: \(\mu = \sum xP(x) \quad \text{ou} \quad \int xf(x)dx\).
  • Variância populacional: \(\sigma^{2} = \sum (x-\mu)^{2}P(x) \quad \text{ou} \quad \int (x-\mu)^{2}f(x)dx\).
  • Proporção populacional: \(p\) é a verdadeira proporção de 1s na população.
  • Mediana populacional: \(\phi_{.5}\) é o valor tal que \(P(X \le \phi_{.5}) \ge .5\) e \(P(X \ge \phi_{.5}) \ge .5\).

Estatística: Um valor numérico calculado a partir de uma amostra \(X_{1},...,X_{n}\).

  • Média amostral: \(\overline{X}=\frac{1}{n}\sum_{i=1}^{n}X_{i}\).
  • Variância amostral: \(S^{2}=\frac{1}{n-1}\sum_{i=1}^{n}(X_{i}-\overline{X})^{2}\).
  • Proporção amostral: \(\hat{p} = \frac{\text{número de 1s}}{n}\).
  • Mediana amostral: \(\hat{\phi}_{.5} = \text{valor central (ou média dos centrais)}\).

2.2 Distribuições Amostrais

O valor de uma estatística varia de amostra para amostra. Por outras palavras, amostras diferentes resultarão em valores diferentes para uma estatística. Portanto, uma estatística é uma variável aleatória com uma distribuição!

Distribuição Amostral: A distribuição dos valores da estatística provenientes de todas as amostras possíveis de tamanho \(n\).

O Método de ‘Força Bruta’ para construir uma Distribuição Amostral

Para entender a origem de uma distribuição amostral, imagine o seguinte processo exato (ou de “força bruta”):

  1. Retirar todas as amostras possíveis de tamanho \(n\) da população.
  2. Calcular o valor da estatística (por exemplo, a média \(\overline{X}\)) para cada uma dessas amostras.
  3. Exibir a distribuição dos valores dessa estatística na forma de uma tabela, de um gráfico (histograma) ou de uma equação.

Na prática, como as populações costumam ser imensas (ou infinitas), é impossível calcular todas as amostras possíveis. É por isso que dependemos de teoremas matemáticos (como o Teorema Central do Limite) para deduzir o formato dessa distribuição sem precisarmos usar a força bruta!

2.2.1 Distribuição Amostral da Média (\(\overline{X}\))

Na estatística inferencial, é comum utilizar a estatística \(\overline{X}\) para estimar \(\mu\).

Média e Variância: Para qualquer tamanho de amostra \(n\) e uma Amostra Aleatória Simples (AAS) proveniente de qualquer distribuição populacional com média \(\mu_{x}\) e variância \(\sigma_{x}^{2}\):

\[E(\overline{X})=\mu_{x}\left(E\left(\sum_{i=1}^n X_i\right) = n \mu_{x}\right) \quad \text{e} \quad Var(\overline{X})=\frac{\sigma_{x}^{2}}{n}\left(E\left(\sum_{i=1}^n X_i\right) = n \sigma^2_{x}\right).\] Para compreendermos como a média amostral se comporta, utilizamos as seguintes propriedades das combinações lineares:

Teorema 1: Combinações Lineares

Sejam \(a_{i}\) para \(i=1,2,...,k\) constantes e sejam \(X_{i}\) para \(i=1,2,...,k\) variáveis aleatórias. Então:

  1. Esperança: \(E\left(\sum_{i=1}^{k}a_{i}X_{i}\right) = \sum_{i=1}^{k}a_{i}E(X_{i})\)
  2. Variância: \(Var\left(\sum_{i=1}^{k}a_{i}X_{i}\right) = \sum_{i=1}^{k}a_{i}^{2}Var(X_{i})\) (se \(X_{1},...,X_{k}\) forem mutuamente independentes).

Quando os dados são normais: Para uma AAS proveniente de uma distribuição normal \(N(\mu_{x},\sigma_{x}^{2})\): \[\overline{X} \sim N\left(\mu_{x}, \frac{\sigma_{x}^{2}}{n}\right).\]

Exemplo: Suponha que os níveis de colesterol de homens adultos sigam uma distribuição \(N(210\,\text{mg/dL}, (37\,\text{mg/dL})^2)\).

  1. Indique um intervalo centrado na média \(\mu\) que capture os 95% centrais de todos os valores de colesterol.

  2. Indique a distribuição amostral de \(\overline{X}\), a média amostral dos valores de colesterol obtidos a partir de AAS (Amostras Aleatórias Simples) de tamanho \(n = 10\).

  3. Indique um intervalo centrado na média \(\mu\) que capture os 95% centrais de todos os valores de média amostral de colesterol obtidos a partir de AAS de tamanho \(n = 10\).

Para amostras de grandes dimensões (Teorema Central do Limite):

Para um grande tamanho de amostra \(n\) e uma AAS proveniente de qualquer distribuição populacional com variância finita \(\sigma_{x}^{2} < \infty\), as distribuições amostrais aproximadas são:

\[\overline{X} \dot{\sim} N\left(\mu_{x}, \frac{\sigma_{x}^{2}}{n}\right) \quad \text{e} \quad \sum\limits_{i=1}^n X_i \dot{\sim} N\left(n\mu_{x}, n\sigma_{x}^{2}\right).\] Este último resultado é consequência do famoso Teorema Central do Limite.

Sejam \(X_{1},\dots,X_{n}\) uma AAS de uma distribuição com média \(\mu_x\) e variância \(\sigma^2_x < \infty\). Então a distribuição de

\[U_n = \frac{\overline{X} - \mu_x}{\sigma_x/\sqrt{n}},\] converge para \(N(0,1)\) a medida que \(n \to \infty\). Provaremos esse teorema mais à frente.

Exemplo Importante (Ensaios de Bernoulli):

Seja \(X = \left\{ \begin{aligned} 1 & \quad\text{se}\; \underline{\hspace{2cm}}\; \text{com probabilidade}\; p: \underline{\hspace{2cm}}\\ 0 & \quad\text{se}\; \underline{\hspace{2cm}}\; \text{com probabilidade}\; 1-p: \underline{\hspace{2cm}} \end{aligned} \right.\)

Então, dizemos que \(X\) segue uma distribuição de Bernoulli: \[X \sim \text{Bern}(p) = \text{Bin}(n = 1, p).\]

  1. Desenhe uma representação gráfica da função de massa de probabilidade de \(X\).

  2. Encontre \(E(X)\) e \(Var(X)\).

  3. Suponha que foi recolhida uma Amostra Aleatória Simples (AAS) \(X_1, X_2, \dots, X_{40}\). Indique a distribuição amostral aproximada de \(\overline{X}\) (geralmente denotada por \(\hat{p} = \overline{X}\), o que indica que \(\overline{X}\) é uma proporção amostral).

2.2.2 Distribuição Amostral da Variância (\(S^{2}\))

Um parâmetro populacional de interesse comum é a variância populacional \(\sigma^2\). Na estatística inferencial, é comum usar a estatística \(S^2\) para estimar \(\sigma^2\). Portanto, a distribuição amostral de \(S^2\) é de grande interesse.

Distribuição Qui-quadrado (\(\chi^2\))

A soma dos quadrados de variáveis normais padrão independentes é distribuída como uma variável aleatória \(\chi^2\). Mais formalmente:

  • Se \(Z_1, \dots, Z_\nu\) são independentes e distribuídos como \(N(0, 1)\), então:

\[\sum_{i=1}^{\nu} Z_i^2 \sim \chi^2(\nu),\]

em que \(\chi^2(\nu)\) é chamada de distribuição qui-quadrado com \(\nu\) graus de liberdade.

  • Para qualquer tamanho de amostra \(n\) e uma AAS \(X_1, X_2, \dots, X_n\) de uma distribuição normal \(N(\mu_x, \sigma_x^2)\): \[\sum_{i=1}^{n} \left(\frac{X_i - \mu_x}{\sigma_x} \right)^2 \sim \chi^2(n)\]

Nota: Use a Tabela do SIGAA para cálculos de probabilidade.

2.2.2.1 Média e Variância

Se \(C \sim \chi^2(\nu)\), então: \[ \left\{ \begin{aligned} E(C) &= \nu; \\ Var(C) &= 2\nu. \end{aligned} \right. \]

Para qualquer tamanho de amostra \(n > 1\) e uma AAS \(X_1, X_2, \dots, X_n\) de qualquer distribuição populacional com média \(\mu_x\) e variância \(\sigma_x^2\):

  • \(E(S^2) = \sigma_x^2.\)
  • \(Var(S^2) = \frac{2\sigma_x^4}{n-1}.\)

2.2.2.2 Distribuição Amostral quando os dados são Normais

Para qualquer tamanho de amostra \(n > 1\) e uma AAS \(X_1, X_2, \dots, X_n\) originada de uma distribuição normal \(N(\mu_x, \sigma_x^2)\):

\[\frac{(n - 1)S^2}{\sigma_x^2} \sim \chi^2(n - 1).\]

2.2.3 Distribuição Amostral de \(\frac{\overline{X}-\mu}{S/\sqrt{n}}\)

Na estatística inferencial, a estatística de teste \(\frac{\bar{X} - \mu}{S/\sqrt{n}}\) é frequentemente utilizada para determinar a quantos erros padrões (\(S/\sqrt{n}\)) a média amostral \(\bar{X}\) está de um valor hipotético \(\mu\). Portanto, a distribuição amostral desta razão é de extrema importância.

Distribuição t de Student

Definição 2.1: Se \(Z \sim N(0, 1)\), \(W \sim \chi^2(\nu)\), e \(Z\) e \(W\) são independentes, então:

\[T = \frac{Z}{\sqrt{W/\nu}} \sim t(\nu),\]

em que \(t(\nu)\) é chamada de distribuição t com \(\nu\) graus de liberdade.

Se \(T \sim t(\nu)\), então:

  • \(E(T) = 0\) para \(\nu > 1\).
  • \(Var(T) = \frac{\nu}{\nu - 2}\) para \(\nu > 2\).

2.2.3.1 Distribuição Amostral quando os dados são Normais

Para qualquer tamanho de amostra \(n > 1\) e uma AAS \(X_1, X_2, \dots, X_n\) de uma \(N(\mu, \sigma^2)\):

  • \(\bar{X}\) e \(S^2\) são independentes.
  • Pelos Teoremas 2.1, 2.3 e Definição 2.1:

\[\frac{\sqrt{n}(\bar{X} - \mu)/\sigma}{\sqrt{\frac{(n-1)S^2/\sigma^2}{n-1}}} = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n - 1).\]

2.2.3.2 Distribuição Amostral para Grandes Amostras

Para um tamanho de amostra GRANDE \(n\) e uma AAS \(X_1, X_2, \dots, X_n\) de qualquer distribuição populacional com média \(\mu_x\) e variância \(\sigma_x^2 < \infty\):

\[T = \frac{\bar{X} - \mu_x}{S/\sqrt{n}}\, \dot{\sim} \, t(n - 1).\]

Fatos Úteis
  • PDF de T: A função densidade de probabilidade é dada por:

\[f(t) = \frac{\Gamma(\frac{\nu+1}{2})}{\Gamma(\frac{\nu}{2})\sqrt{\nu\pi}} \left( 1 + \frac{t^2}{\nu} \right)^{-\frac{\nu+1}{2}}.\]

  • A distribuição \(t\) é simétrica em torno de 0 e possui formato de sino como a \(N(0, 1)\), porém com caudas mais pesadas (mais grossas).

  • Conforme \(\nu \to \infty\), a distribuição \(t(\nu)\) aproxima-se da distribuição normal padrão.

  • Use a Tabela do SIGAA para cálculos de probabilidade.

2.2.3.3 Exemplos

Suponha que os níveis de colesterol em homens adultos sejam distribuídos como \(N(210 \text{ mg/dL}, \sigma^2)\).

  1. Forneça a distribuição amostral de \(\frac{\bar{X} - \mu}{S/\sqrt{n}}\), onde as estatísticas \(\bar{X}\) e \(S^2\) são calculadas a partir de uma AAS de tamanho \(n = 10\).

  2. Se \(S^2 = 36,5^2\), forneça um intervalo centrado na média \(\mu\) que capture os 95% centrais de todos os valores de médias amostrais de colesterol obtidos de AAS de tamanho \(n = 10\).

2.2.4 Distribuição Amostral do Razão de Variâncias

Na estatística inferencial, frequentemente o interesse recai sobre a comparação das variâncias \(\sigma_1^2\) e \(\sigma_2^2\) de duas populações distintas para determinar se são diferentes. Com base em duas Amostras Aleatórias Simples (AAS), uma de tamanho \(n_1\) com variância amostral \(S_1^2\) e outra de tamanho \(n_2\) com variância amostral \(S_2^2\), a estatística \(\frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2}\) é utilizada.

Distribuição F

Definição 2.2: Se \(W_1 \sim \chi^2(\nu_1)\) e \(W_2 \sim \chi^2(\nu_2)\) são independentes, então:

\[F = \frac{W_1/\nu_1}{W_2/\nu_2} \sim F(\nu_1, \nu_2).\]

A distribuição \(F(\nu_1, \nu_2)\) possui \(\nu_1\) graus de liberdade no numerador e \(\nu_2\) graus de liberdade no denominador.

Se \(F \sim F(\nu_1, \nu_2)\), então:

\[ \left\{ \begin{aligned} E(F) &= \frac{\nu_2}{\nu_2 - 2}, \quad \text{para}\; \nu_2 > 2; \\ Var(F) &= \frac{2\nu_2^2 (\nu_1 + \nu_2 - 2)}{\nu_1 (\nu_2 - 2)^2 (\nu_2 - 4)}, \quad \text{para}\; \nu_2 > 4. \end{aligned} \right. \]

2.2.4.1 Distribuição Amostral quando os dados são Normais

Se \(X_1, \dots, X_{n_1}\) é uma AAS de \(N(\mu_1, \sigma_1^2)\) e \(Y_1, \dots, Y_{n_2}\) é uma AAS independente de \(N(\mu_2, \sigma_2^2)\), então as variáveis qui-quadrado são independentes:

\[W_1 = \frac{(n_1 - 1)S_1^2}{\sigma_1^2} \sim \chi^2(n_1 - 1) \quad \text{e} \quad W_2 = \frac{(n_2 - 1)S_2^2}{\sigma_2^2} \sim \chi^2(n_2 - 1).\]

Portanto,

\[F = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1 - 1, n_2 - 1).\]

2.2.5 Exemplo

A variância do tempo de reação de mulheres é diferente da variância do tempo de reação de homens? Jason Paulak (Universidade de Cincinnati) realizou um experimento web para responder a isso.

  • Homens (\(n_2 = 469\)): \(\bar{X}_2 = 383,2\) ms, \(S_2 = 335,7\) ms.

  1. Forneça a distribuição amostral de \(F = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2}\):

  2. Se as duas variâncias populacionais forem de fato iguais (\(\sigma_1^2 = \sigma_2^2\)), qual é a probabilidade de observar uma razão de variâncias amostrais igual ou maior que a encontrada?

2.3 Prova do Teorema Central do Limite (TCL)

Teorema do Limite Central

Seja \(X_{1},X_{2},...,X_{n}\) uma AAS de uma distribuição com média \(\mu_{x}\) e variância \(\sigma_{x}^{2}<\infty\). Então:

\[\lim_{n\rightarrow\infty} U_n = \lim_{n\rightarrow\infty}\frac{\overline{X}-\mu_{x}}{\sigma_{x}/\sqrt{n}} = U \sim N(0,1).\]

2.3.0.1 A Prova do TCL

A demonstração baseia-se nas Funções Geradoras de Momentos (FGM). O Teorema XX afirma que cada variável aleatória possui uma FGM única, \(M(t)\), que a identifica. Para uma variável contínua \(Y\) com densidade \(f(y)\), a FGM é:

\[M(t) = E(e^{tY}) = \int_{-\infty}^{\infty} e^{ty} f(y) dy.\]

2.3.1 Roteiro da Prova:

  1. Usar a série de Taylor para encontrar a FGM de \(Z_i = \frac{X_i - \mu}{\sigma}\).

  2. Encontrar a FGM de \(U_n = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}}\).

  3. Mostrar que, conforme \(n \to \infty\), a FGM de \(U_n\) converge para \(e^{t^2/2}\).

  4. Identificar \(e^{t^2/2}\) como a FGM da Normal Padrão \(N(0, 1)\).

2.3.1.1 Passo 1: A variável padronizada \(Z_i\)

Seja \(Z_i = \frac{X_i - \mu}{\sigma}\). Aqui, a suposição de variância finita (\(\sigma^2 < \infty\)) é crucial.

  1. Propriedades: Mostre que \(E(Z) = 0\) e \(Var(Z) = E(Z^2) = 1\). (Dica: Z é uma função linear de X).

  2. Momentos na Origem: Mostre que:

  • \(M_Z(0) = 1\)
  • \(M_Z'(0) = E(Z) = 0\)
  • \(M_Z''(0) = E(Z^2) = 1\)
  1. Expansão de Taylor: Expandindo \(M_Z(t)\) em torno de \(t = 0\) e avaliando em \(t/\sqrt{n}\):

\[M_Z\left(\frac{t}{\sqrt{n}}\right) = 1 + \frac{(t/\sqrt{n})^2}{2!} + R\left(\frac{t}{\sqrt{n}}\right) = 1 + \frac{t^2}{2n} + R\left(\frac{t}{\sqrt{n}}\right).\] em que \(R\left(\frac{t}{\sqrt{n}}\right)\) representa o resto de termos cúbicos e de ordem superior.

  1. Resto: Mostre que o resto \(\lim_{n \to \infty} \frac{R(t/\sqrt{n})}{(t/\sqrt{n})^2} = n \lim_{n \to \infty}R(t/\sqrt{n}) = 0.\)

2.3.1.2 Passo 2: A FGM de \(U_n\)

Podemos escrever \(U_n\) como:

\[U_n = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} = \frac{1}{\sqrt{n}} \sum_{i=1}^{n} Z_i.\]

Como os \(X_i\) são uma AAS (independentes), a FGM da soma é o produto das FGMs:

\[M_{U_n}(t) = \prod_{i=1}^{n} M_{Z_i}\left(\frac{t}{\sqrt{n}}\right) = \left[ M_Z\left(\frac{t}{\sqrt{n}}\right) \right]^n.\]

2.3.1.3 Passo 3: O Limite

Substituindo a expansão de Taylor:

\[M_{U_n}(t) = \left[ 1 + \frac{t^2}{2n} + R\left(\frac{t}{\sqrt{n}}\right) \right]^n.\]

Aplicando o limite \(n \to \infty\) e usando o fato de que \(\lim_{n \to \infty} (1 + \frac{a}{n})^n = e^a\), então:

\[\lim_{n \to \infty} M_{U_n}(t) = e^{t^2/2}.\] (Use o fato que \(\lim_{n \to \infty} \frac{R(t/\sqrt{n})}{(t/\sqrt{n})^2} = n \lim_{n \to \infty}R(t/\sqrt{n}) = 0.\).

2.3.1.4 Passo 4: Reconhecimento da Distribuição

Se \(U \sim N(0, 1)\), então sua FGM é \(M_U(t) = e^{t^2/2}\).

Prova rápida:

\[E(e^{tU}) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} e^{tu} e^{-u^2/2} du.\]

Completando o quadrado no expoente: \(tu - \frac{u^2}{2} = -\frac{1}{2}(u-t)^2 + \frac{t^2}{2}\).

\[E(e^{tU}) = e^{t^2/2} \underbrace{\int_{-\infty}^{\infty} \frac{1}{\sqrt{2\pi}} e^{-\frac{(u-t)^2}{2}} du}_{1 \text{ (integral de uma FDP Normal)}} = e^{t^2/2}.\]
